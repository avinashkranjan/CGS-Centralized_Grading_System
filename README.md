## How to run Locally?

#### Step 1:
    pip install pipenv
#### Step 2:
    pipenv shell
#### Step 3:
    pipenv install -r requirements.txt
#### Step 4:
    cd CGS-Centralized_Grading_System
#### Step 5:    
    python manage.py migrate
#### Step 6:   
    python manage.py createsuperuser
#### Step 7:
    python manage.py runserver
